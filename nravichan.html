How did I build it?

This site is built using HTML5, CSS3 and JQUERY. HTML5 provides cross browser support enabling all popular browsers to support it. CSS3 provides a rich experience for transforming and creating animations. Keyframes in CSS3 helps me create complicated animations by using specific style rule applications at various steps through a period of time. Using Bootstrap allows me to customize and factor in my own needs. Bootstrap is a consistent and responsive framework, which provides a uniform result across browsers.
